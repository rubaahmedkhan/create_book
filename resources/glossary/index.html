Glossary

Key terms and definitions for Physical AI and robotics.

A​

Action - A long-running ROS 2 task with feedback (e.g., navigation to a goal)

Actuator - A mechanical component that produces motion (motors, servos)

D​

DOF (Degrees of Freedom) - Number of independent movements a robot can make

G​

Gazebo - Open-source 3D robot simulator with physics engine

I​

IMU (Inertial Measurement Unit) - Sensor measuring acceleration and rotation

Isaac Sim - NVIDIA's photorealistic robot simulator built on Omniverse

L​

LiDAR - Light Detection and Ranging sensor for 3D environment mapping

N​

Nav2 - ROS 2 navigation framework for autonomous mobile robots

Node - Independent process in ROS 2 that performs computation

P​

Physical AI - AI systems that interact with and learn from the physical world

R​

rclpy - ROS 2 client library for Python

ROS 2 - Robot Operating System 2, middleware for robotic applications

S​

Service - Synchronous request-response communication pattern in ROS 2

Sim-to-Real - Transfer of learned behaviors from simulation to physical robots

T​

Topic - Named bus for asynchronous message passing in ROS 2

U​

URDF - Unified Robot Description Format, XML for robot models

V​

VLA - Vision-Language-Action, multimodal AI for robotics

VSLAM - Visual Simultaneous Localization and Mapping
