# Physical AI & Humanoid Robotics

A comprehensive course on Physical AI and Humanoid Robotics — covering ROS 2, Gazebo simulation, NVIDIA Isaac, and Vision-Language-Action models. Bilingual (English + Urdu).

## How It Works

1. Sign up on the Learning Platform
2. Select your skill level and OS
3. Get a personalized dashboard with links to the book
4. Read the book — content is filtered by your level (Beginner / Intermediate / Advanced)
5. Switch between English and Urdu on any page
