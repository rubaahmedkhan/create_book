# Prerequisites

Before starting this course, you should have:

## Programming Skills

- **Python 3.10+**: Intermediate proficiency
  - Object-oriented programming
  - Async/await patterns
  - Package management with pip

## Mathematical Background

- **Linear Algebra**: Vectors, matrices, transformations
- **Calculus**: Basic derivatives and optimization
- **Probability**: Statistical concepts for robotics

## AI/ML Experience

- Familiarity with neural networks
- Understanding of training pipelines
- Experience with PyTorch or TensorFlow (helpful but not required)

## Development Environment

- **Operating System**: Ubuntu 22.04 LTS (required for ROS 2)
  - Native installation, WSL2, or Docker
- **Text Editor**: VS Code, PyCharm, or similar
- **Version Control**: Git basics

## No Robotics Experience Required!

This course is designed for AI/ML practitioners transitioning to Physical AI. We'll teach all robotics concepts from scratch.
