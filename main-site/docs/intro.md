# Welcome to Physical AI & Humanoid Robotics

A comprehensive educational textbook for a 13-week course that guides students from digital AI to embodied intelligence in physical systems.

## Course Overview

This textbook is designed for students with AI/ML backgrounds who want to learn robotics and Physical AI. Over 13 weeks, you'll master the tools and techniques needed to build intelligent robotic systems.

### Four Core Modules

1. **Module 1: ROS 2 Fundamentals** - The Robotic Nervous System
   - ROS 2 architecture and core concepts
   - Nodes, topics, services, and actions
   - Building ROS 2 packages with Python

2. **Module 2: Gazebo & Unity Simulation** - The Digital Twin
   - URDF robot description formats
   - Physics simulation with Gazebo
   - High-fidelity rendering with Unity

3. **Module 3: NVIDIA Isaac Platform** - The AI-Robot Brain
   - NVIDIA Isaac Sim for photorealistic simulation
   - Isaac ROS for hardware-accelerated VSLAM
   - Nav2 path planning for bipedal locomotion

4. **Module 4: Vision-Language-Action (VLA)** - Multimodal Integration
   - OpenAI Whisper for voice command processing
   - LLM-based cognitive planning with GPT-4
   - **Capstone Project**: Autonomous humanoid robot

## Learning Approach

This textbook follows a **simulation-first pedagogy**:

1. **See it work** - Watch simulations and demonstrations
2. **Understand why** - Learn the theoretical foundations
3. **Build it yourself** - Hands-on labs and exercises
4. **Deploy it** - Sim-to-real transfer techniques

## Prerequisites

- **Programming**: Intermediate Python (3.10+)
- **Math**: Linear algebra, basic calculus
- **AI/ML**: Familiarity with neural networks and training
- **No robotics experience required!**

## Hardware Requirements

See [Hardware Requirements](./getting-started/hardware-requirements) for detailed specifications. The course offers three paths:

- **Edge Deployment**: NVIDIA Jetson Orin Nano (~$250)
