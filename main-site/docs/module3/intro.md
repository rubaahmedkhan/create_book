# Module 3: NVIDIA Isaac Platform

**The AI-Robot Brain** | Weeks 7-10

Leverage NVIDIA Isaac Sim and Isaac ROS for advanced robotic AI applications with hardware acceleration.

## What You'll Build

- Photorealistic robot simulations in Isaac Sim
- Hardware-accelerated VSLAM with Isaac ROS
- Nav2 path planning for bipedal humanoid robots
- Sim-to-real transfer pipelines

## Learning Outcomes

1. Use Isaac Sim for photorealistic robot simulation
2. Implement Isaac ROS for GPU-accelerated perception
3. Apply Nav2 for autonomous navigation
4. Master sim-to-real transfer techniques

## Prerequisites

- ✅ Module 1 & 2 completed
- ✅ RTX GPU (3060+) or cloud instance
- ✅ CUDA 12.0+ installed

Start with [Week 7-8: Isaac Sim Introduction](./week7-8/isaac-sim-intro).
