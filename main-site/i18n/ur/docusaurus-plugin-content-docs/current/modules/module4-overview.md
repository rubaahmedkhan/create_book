# ماڈیول 4: Vision-Language-Action (VLA)

**ملٹی موڈل انٹیگریشن اور کیپ اسٹون** - ہفتے 11-13

## جائزہ

واقعی ذہین روبوٹک سسٹمز کے لیے بصارت، زبان، اور عمل کو ضم کریں۔

## سیکھنے کے نتائج

- OpenAI Whisper کے ساتھ آواز کے احکامات پر کارروائی
- GPT-4 کے ساتھ LLM پر مبنی علمی منصوبہ بندی کو نافذ کرنا
- آواز سے عمل تک کی پائپ لائنز بنانا
