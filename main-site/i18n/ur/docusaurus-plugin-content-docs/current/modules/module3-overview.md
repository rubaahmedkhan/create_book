# ماڈیول 3: NVIDIA Isaac پلیٹ فارم

**AI-روبوٹ دماغ** - ہفتے 7-10

## جائزہ

جدید روبوٹک AI ایپلیکیشنز کے لیے NVIDIA Isaac Sim اور Isaac ROS کا فائدہ اٹھائیں۔

## سیکھنے کے نتائج

- حقیقت پسندانہ سمیولیشن کے لیے Isaac Sim کا استعمال
- ہارڈویئر ایکسلریٹڈ VSLAM اور نیویگیشن کے لیے Isaac ROS کو نافذ کرنا
- دو پیروں والی حرکت کے لیے Nav2 پاتھ پلاننگ کا اطلاق
- Sim-to-real ٹرانسفر تکنیکوں میں مہارت حاصل کرنا
