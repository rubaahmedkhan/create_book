# ماڈیول 2: Gazebo اور Unity سمیولیشن

**ڈیجیٹل ٹون** - ہفتے 4-6

## جائزہ

Gazebo کے ساتھ فزکس اور Unity کے ساتھ اعلیٰ معیار کی رینڈرنگ کے لیے روبوٹ سمیولیشن میں مہارت حاصل کریں۔

## سیکھنے کے نتائج

- URDF اور SDF روبوٹ تفصیلات بنانا
- Gazebo کے ساتھ فزکس کی سمیولیشن کرنا
- حقیقت پسندانہ تصویر کاری کے لیے Unity کو ضم کرنا
- سینسرز کی سمیولیشن کرنا (LiDAR، کیمرے، IMUs)
