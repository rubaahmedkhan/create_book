# ماڈیول 1: ROS 2 بنیادی باتیں

**روبوٹک اعصابی نظام** - ہفتے 1-3

## جائزہ

Robot Operating System 2 (ROS 2) سیکھیں، جو روبوٹک ایپلیکیشنز کے لیے انڈسٹری معیاری مڈل ویئر ہے۔

## سیکھنے کے نتائج

- ROS 2 آرکیٹیکچر اور بنیادی تصورات کو سمجھنا
- ROS 2 نوڈز بنانا اور منظم کرنا
- پبلشرز، سبسکرائبرز، سروسز، اور ایکشنز کو نافذ کرنا
- Python (rclpy) کے ساتھ ROS 2 پیکجز بنانا
- لانچ فائلز اور پیرامیٹر مینجمنٹ کا استعمال

## موضوعات کا احاطہ

1. ROS 2 آرکیٹیکچر
2. نوڈز اور کمیونیکیشن پیٹرنز
3. ROS 2 پیکجز بنانا
4. لانچ فائلز اور پیرامیٹرز
5. عملی لیبز
