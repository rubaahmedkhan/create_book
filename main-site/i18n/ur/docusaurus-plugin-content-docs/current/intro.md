# فزیکل اے آئی اور ہیومنائیڈ روبوٹکس میں خوش آمدید

13 ہفتوں کے کورس کے لیے ایک جامع تعلیمی کتاب جو طلباء کو ڈیجیٹل اے آئی سے لے کر فزیکل سسٹمز میں مجسم ذہانت تک رہنمائی فراہم کرتی ہے۔

## کورس کا جائزہ

یہ کتاب ان طلباء کے لیے ڈیزائن کی گئی ہے جن کے پاس AI/ML کا پس منظر ہے اور وہ روبوٹکس اور فزیکل اے آئی سیکھنا چاہتے ہیں۔ 13 ہفتوں میں، آپ ذہین روبوٹک سسٹم بنانے کے لیے درکار ٹولز اور تکنیکوں میں مہارت حاصل کریں گے۔

### چار بنیادی ماڈیولز

1. **ماڈیول 1: ROS 2 بنیادی باتیں** - روبوٹک اعصابی نظام
   - ROS 2 آرکیٹیکچر اور بنیادی تصورات
   - نوڈز، ٹاپکس، سروسز، اور ایکشنز
   - Python کے ساتھ ROS 2 پیکجز بنانا

2. **ماڈیول 2: Gazebo اور Unity سمیولیشن** - ڈیجیٹل ٹون
   - URDF روبوٹ تفصیل کی شکلیں
   - Gazebo کے ساتھ فزکس سمیولیشن
   - Unity کے ساتھ اعلیٰ معیار کی رینڈرنگ

3. **ماڈیول 3: NVIDIA Isaac پلیٹ فارم** - AI-روبوٹ دماغ
   - حقیقت پسندانہ سمیولیشن کے لیے NVIDIA Isaac Sim
   - ہارڈویئر ایکسلریٹڈ VSLAM کے لیے Isaac ROS
   - دو پیروں والی حرکت کے لیے Nav2 پاتھ پلاننگ

4. **ماڈیول 4: Vision-Language-Action (VLA)** - ملٹی موڈل انٹیگریشن
   - آواز کمانڈ پروسیسنگ کے لیے OpenAI Whisper
   - GPT-4 کے ساتھ LLM پر مبنی علمی منصوبہ بندی
   - **کیپ اسٹون پروجیکٹ**: خودکار ہیومنائیڈ روبوٹ

## سیکھنے کا طریقہ

یہ کتاب **سمیولیشن فرسٹ تدریسی طریقہ** کی پیروی کرتی ہے:

1. **اسے کام کرتے دیکھیں** - سمیولیشنز اور مظاہروں کو دیکھیں
2. **سمجھیں کیوں** - نظریاتی بنیادوں کو سیکھیں
3. **اسے خود بنائیں** - ہینڈز آن لیبز اور مشقیں
4. **اسے تعینات کریں** - Sim-to-real ٹرانسفر تکنیک

## پیشگی ضروریات

- **پروگرامنگ**: انٹرمیڈیٹ Python (3.10+)
- **ریاضی**: لکیری الجبرا، بنیادی کیلکولس
- **AI/ML**: نیورل نیٹ ورکس اور ٹریننگ سے واقفیت
- **کوئی روبوٹکس کا تجربہ ضروری نہیں!**

## ہارڈویئر کی ضروریات

تفصیلی وضاحتوں کے لیے [ہارڈویئر کی ضروریات](./getting-started/hardware-requirements) دیکھیں۔ کورس تین راستے پیش کرتا ہے:

- **لوکل ڈیولپمنٹ**: RTX 3060+ ورک اسٹیشن
- **کلاؤڈ انسٹنسز**: AWS g5.2xlarge (~$200-250/سہ ماہی)
- **ایج ڈپلائمنٹ**: NVIDIA Jetson Orin Nano (~$250)

## شروع کریں

فزیکل اے آئی میں اپنا سفر شروع کرنے کے لیے تیار ہیں؟ اپنے ڈیولپمنٹ ماحول کو سیٹ اپ کرنے کے لیے [شروع کرنا](./getting-started/prerequisites) سے شروع کریں۔
