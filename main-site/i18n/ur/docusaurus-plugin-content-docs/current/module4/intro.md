# ماڈیول 4: Vision-Language-Action (VLA)

**ملٹی موڈل انٹیگریشن اور کیپ اسٹون** | ہفتہ 11-13

واقعی ذہین روبوٹک سسٹمز کے لیے ویژن، زبان، اور ایکشن کو یکجا کریں۔ اپنا خودکار ہیومنائیڈ روبوٹ کیپ اسٹون پروجیکٹ بنائیں۔

## آپ کیا بنائیں گے

- OpenAI Whisper کے ساتھ آواز کمانڈ پروسیسنگ
- GPT-4 کے ساتھ LLM پر مبنی علمی منصوبہ بندی
- آواز سے ایکشن پائپ لائنز
- **کیپ اسٹون پروجیکٹ**: ملٹی موڈل انٹیگریشن کے ساتھ خودکار ہیومنائیڈ روبوٹ

## سیکھنے کے نتائج

1. OpenAI Whisper کے ساتھ آواز کمانڈز پروسیس کرنا
2. GPT-4 کے ساتھ LLM پر مبنی ٹاسک پلاننگ نافذ کرنا
3. مکمل آواز سے ایکشن پائپ لائنز بنانا
4. تمام ماڈیولز کو ایک فعال ہیومنائیڈ روبوٹ میں یکجا کرنا

## پیشگی ضروریات

- ✅ ماڈیولز 1-3 مکمل
- ✅ OpenAI API کی
- ✅ LLMs اور ٹرانسفارمرز کی سمجھ

## ماڈیول کا ڈھانچہ

### ہفتہ 11: آواز پروسیسنگ
- OpenAI Whisper سیٹ اپ
- آڈیو کیپچر اور پروسیسنگ
- آواز پہچان نافذ کرنا
- **لیب**: وائس کمانڈ سسٹم

### ہفتہ 12: LLM انٹیگریشن
- GPT-4 کے ساتھ ٹاسک پلاننگ
- آواز سے ایکشن پائپ لائن
- ROS 2 کے ساتھ انٹیگریشن
- **لیب**: ذہین روبوٹ کنٹرولر

### ہفتہ 13: کیپ اسٹون پروجیکٹ
- تمام ماڈیولز یکجا کرنا
- ٹیسٹنگ اور ڈیبگنگ
- پروجیکٹ پریزنٹیشن
- **کیپ اسٹون**: خودکار ہیومنائیڈ روبوٹ
