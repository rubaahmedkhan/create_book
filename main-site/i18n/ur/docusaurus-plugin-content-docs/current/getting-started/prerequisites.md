# پیشگی ضروریات

اس کورس کو شروع کرنے سے پہلے، آپ کے پاس یہ ہونا چاہیے:

## پروگرامنگ کی مہارتیں

- **Python 3.10+**: انٹرمیڈیٹ مہارت
  - آبجیکٹ اورینٹڈ پروگرامنگ
  - Async/await پیٹرنز
  - pip کے ساتھ پیکج مینجمنٹ

## ریاضی کا پس منظر

- **لکیری الجبرا**: ویکٹرز، میٹرکسز، تبدیلیاں
- **کیلکولس**: بنیادی ڈیریویٹوز اور آپٹیمائزیشن
- **احتمال**: روبوٹکس کے لیے شماریاتی تصورات

## AI/ML کا تجربہ

- نیورل نیٹ ورکس سے واقفیت
- ٹریننگ پائپ لائنز کی سمجھ
- PyTorch یا TensorFlow کا تجربہ (مددگار لیکن ضروری نہیں)

## ڈیولپمنٹ ماحول

- **آپریٹنگ سسٹم**: Ubuntu 22.04 LTS (ROS 2 کے لیے ضروری)
  - نیٹو انسٹالیشن، WSL2، یا Docker
- **ٹیکسٹ ایڈیٹر**: VS Code، PyCharm، یا اسی طرح کا
- **ورژن کنٹرول**: Git کی بنیادی باتیں

## کوئی روبوٹکس کا تجربہ ضروری نہیں!

یہ کورس AI/ML پریکٹیشنرز کے لیے ڈیزائن کیا گیا ہے جو فزیکل اے آئی کی طرف منتقل ہو رہے ہیں۔ ہم تمام روبوٹکس تصورات شروع سے سکھائیں گے۔
