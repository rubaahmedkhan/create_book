# ہارڈویئر کی ضروریات

اس کورس میں ماڈیول کے لحاظ سے مختلف کمپیوٹیشنل ضروریات ہیں۔

## تین ڈپلائمنٹ راستے

### 1. لوکل ڈیولپمنٹ (تجویز کردہ)

| جزو | کم از کم | تجویز کردہ |
|-----------|---------|-------------|
| **GPU** | RTX 3060 (8GB VRAM) | RTX 4070 Ti (12GB VRAM) |
| **CPU** | Intel i7 13th Gen | AMD Ryzen 9 |
| **RAM** | 32 GB DDR5 | 64 GB DDR5 |
| **OS** | Ubuntu 22.04 LTS | Ubuntu 22.04 LTS |
| **Storage** | 250 GB SSD | 500 GB NVMe |

### 2. کلاؤڈ انسٹنسز

- **AWS g5.2xlarge**: تقریباً $1.00/گھنٹہ
- **تخمینی لاگت**: $200-250/سہ ماہی (10 گھنٹے/ہفتہ)
- **Ubuntu 22.04 LTS** ضروری

### 3. ایج ڈپلائمنٹ

- **NVIDIA Jetson Orin Nano** (8GB): تقریباً $250
- ماڈیول 3-4 کے لیے (Isaac Sim, VLA)
- کم کارکردگی لیکن سیکھنے کے لیے اچھا

## ماڈیول کے لحاظ سے ضروریات

- **ماڈیول 1-2**: انٹیگریٹڈ گرافکس پر چل سکتا ہے (محدود)
- **ماڈیول 3**: RTX GPU یا کلاؤڈ انسٹنس ضروری
- **ماڈیول 4**: ریئل ٹائم VLA کے لیے RTX GPU کی سخت سفارش
