# Module 2: Gazebo & Unity Simulation

**The Digital Twin** | Weeks 4-6

Master robot simulation with Gazebo for physics-accurate testing and Unity for photorealistic visualization.

## What You'll Build

- URDF robot models with joints, links, and sensors
- Gazebo simulation worlds with physics
- Unity integration for high-fidelity rendering
- Sensor simulations (LiDAR, cameras, IMUs)

## Learning Outcomes

1. Create URDF and SDF robot descriptions
2. Simulate physics with Gazebo
3. Integrate Unity for photorealistic visualization
4. Implement sensor models for perception testing

## Prerequisites

- ✅ Module 1: ROS 2 Fundamentals completed
- ✅ Basic understanding of XML
- ✅ Gazebo Garden installed

Start with [Week 4: URDF Basics](./week4/urdf-basics).
